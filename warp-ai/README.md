# Warp AI - Private BatCave AI Agent

Private, local AI chat assistant in Warp terminal for BatCave privacy/VPN/threat intel.

## Quick Setup
1. Ensure Ollama running: `ollama serve`
2. Pull model: `./setup-ollama.ps1` (pulls llama3)
3. Import agent in Warp: Cmd/Ctrl + , → AI → Agents → Import from [WarpAI-Agent.md]
4. Chat: Cmd/Ctrl + i (Agent chat)

## Features
- VPN troubleshooting (WireGuard/Tor)
- Privacy advice (telemetry block, PII strip)
- Threat intel (strategic analysis)
- BatCave spec queries

Local/private: Uses Ollama (no cloud).
